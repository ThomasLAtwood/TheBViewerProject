Installing and Initializing BViewer

Install the Software

To install BViewer, you can go to the NIOSH chest radiography web page. From there you can download the latest version of the BViewer setup.exe installation file.

You install BViewer by double-clicking on the setup.exe installation file, which you will probably find in your Downloads folder. If your computer account does not have Administrator privileges, you will need someone with administrator access to authorize the installation.



Start BViewer

When the installation is complete, you should see the BViewer icon in your computer desktop window. You can start BViewer by double-clicking your mouse over the BViewer icon.



Enter Your Reader Information

If this is a new installation, BViewer will open by displaying the Set Up BViewer screen. You should immediately enter your reader information in the indicated edit boxes. This information will be automatically included in the appropriate places in your interpretation reports.

The login name and password you enter here will be used to control access to BViewer. As you enter these fields, be sure to write them down and keep them in a safe place. You won't want to be locked out of your own BViewer!



Install the ILO Standard Images

For you to be able to use the BViewer software, you will have to acquire a license to use the ILO standard comparison images. After acquiring this license, you will receive a copy of the ILO standards installation media. You will need to install these standard reference images so BViewer will be able to access them.



Specify Your Local Dicom AE_TITLE

Specifying your Local Dicom Name is important. This field is the Dicom AE_TITLE, which must be used by remote image sending locations for routing images to your BViewer. Multiple users can share a single BViewer workstation. The AE_TITLE to which the images are sent will determine which user gets to view them.

The default AE_TITLE when you first install BViewer is just "BViewer", which works fine for a single user. You can change this if you like. For multiple users, each user should have a different AE_TITLE. This AE_TITLE field is 16 characters or less in length, and may not include any spaces, although it may contain underscore characters.



Specify Your Local Dicom Network Address

In order for remote sites to be able to transmit Dicom image files for you to read, you have to set up a network address for your BViewer workstation. This is usually a job for your local computer technician. To enter the network address, click on the "Set Local Network Address" button. In the edit box that appears, enter the static IP address for your workstation (localhost should do the job), followed by a colon, followed by the port number to be used by the BViewer software to respond to Dicom requests over the network. An example would be localhost:105. This port number should differ from that of any other Dicom device or software active on the same computer.

Any image sending software application or device, such as a CR device or a PACS, which will need to send (over the computer network or internet) images for you to read must have a Dicom routing rule added to specify your network address. This specification must include (a) an externally accessible IP address for reaching your BViewer workstation, (b) the Dicom port number assigned to your BViewer software, and (c) your AE_TITLE.

Note that the BViewer workstation must be accessible from the site that is sending the images. If the sender is external to your local network, the IP address will need to be the externally accessible one, and the port number will have to be forwarded through routers and firewalls between the outside world and your workstation. For example, you may need to add an exception for this port number in your Windows Firewall manager in order to make the port accessible from other locations. This can be resolved by your local computer technician.



Verify BRetriever Is Running

The BRetriever component of the BViewer software runs in the background as a Windows service. It runs whenever your computer is turned on, so it can receive image files even when you are not running BViewer. The BRetriever service is automatically installed and started when you install BViewer. You can use the "Control BRetriever" button on the "Set Up BViewer" tab to check on the status of BRetriever and to stop and start it if something goes wrong. Click on this button and the BRetriever control screen will appear. Click on the "Show Status" button and you should see the message, "Status Check: BRetriever is running", at the bottom of the window. This window displays the BRetriever log file, and can be used to help diagnose any BRetriever problems.

Click on the "Exit" button to close the BViewer control screen. (Since BRetriever is a Windows service, your computer technician can also use the Windows Service Manager to control BRetriever.)

If you have problems getting BRetriever to stop or start, you can fully reset it by restarting your computer.



Log on to BViewer

Whenever you start BViewer after you have entered your reader identification, you will be prompted for your login name and password. Then, if you have entered your credentials correctly, left clicking the mouse on the Log In button will grant you access to BViewer and start it running.
If you choose not to enter a login name in your reader identification data, the login screen will not appear and BViewer will immediately become active. CAUTION: Although configuring BViewer without a login name may be convient, it will allow anyone to access your BViewer data and studies. Likewise, not specifying a secure password may make your BViewer information and studies less secure from prying eyes. For maximum security, you should enter both a login name and a secret password.

Add a New User

To add a second user to BViewer, check the Add New User button during your login or alternatively on the Set Up BViewer panel. An additional user may be added only with the approval of an existing user. After a successful login with the Add New User button checked, the Reader Identification screen will pop up, providing for the entry of the information for the new user.


Specify the Display Parameters for Each Display

Before you begin reading studies, you must enter three parameters that describe your display devices. You must do this separately for each of the displays.

For each display, measure the effective screen height and width in millimeters, using a ruler. Enter these measurements on the Set Up BViewer tab. These measurements are used to display the image at "full size", the estimated size of the original scanned digital image or scanned film.

For each display, enter the grayscale bit depth supported by that display. An ordinary color display typically used as a Windows desktop normally supports a grayscale bit depth of 8. This means that there are 8 bits available for describing the value of a single pixel. An 8-bit display can support a maximum of 256 different shades of gray.

Although an 8-bit display can be used in an emergency for viewing reference images and subject study images, normal interpretation requires a grayscale resolution of at least 10 bits. This supports 1024 different shades of gray, near the ability of the human eye to distinguish adjacent shades of gray.

Some high resolution displays may support 12 or even 16 bits of grayscale. Be sure to enter the appropriate bit depth for each of your displays. (For example, if you mistakenly enter 8 for a 10-bit display, it will only display images using 256 gray values, instead of the 1024 gray values it is designed to render.)





Create a Digital Signature File

On the lower left part of the second page of the report there is a place for a digital signature. If this were a paper report, you would sign it in this space. This section describes how you can produce a digital signature image file that can be used by BViewer to "sign" your reports. If you are not proficient in manipulating files on a computer, you should get the assistance of someone who is.

As a brief summary, here are the steps:

  • 1. Sign your name on a piece of paper and scan it into a JPG image file.
  • 2. Open this file with the Windows Paint program.
  • 3. Using the Paint cropping tool, crop out the region of the signature from the full page image.
  • 4. Save the cropped signature image as a bitmap (BMP) file.
Now, let's look at each of these steps in detail.

Step 1: Scan your signature into a JPG file

This step requires that you have access to some type of scanner that can scan a piece of paper and save it as a JPG file. Some models of "all-in-one" printers have this capability. Depending on the type of scanner, (a) you would insert a USB memory flash drive into the scanner to receive the scanned image, or (b) from a network scanner the image may be sent back to your workstation and saved on your hard drive, for example as an icon on your "desktop".

If you do not have access to a scanner, you may try to open a new image in the Windows Paint program, and use the drawing tool to produce an image of your signature. The cumbersome movements of a typical computer mouse make it somewhat difficult to produce a properly flowing signature using this approach. (It is possible to obtain a type of specialized mouse that can be held like a pen. You can ask around if anybody has one.)

Step 2: Open the signature file in the Windows Paint Program

If you can see your signature image file icon on the Windows desktop, you can click the right mouse button on it to get a pop-up menu. Select "Open With", then select the Paint program.

Another way to open the file is to start the Paint program. For later versions of Windows, you can just type in "Paint" and Windows will find it for you. Otherwise, from the Start button, display the list of programs, then select "Accessories". The Paint program is one of the Windows accessory programs. Select it. When the Paint program starts up, Click on the File menu, then click Open. Use the open file window to locate your signature file. After you have selected your file in the window, click on the "Open" button to open the file.

When the signature image file has been opened in the Paint program, you should see your signature displayed. You may have to scroll the image to find it, depending upon the part of the page where you wrote it. Also, you may have to enlarge the Paint window to see your entire image.

Step 3: Crop out the signature.

Use the rectangular selection tool ("Cropping Tool" indicated in the image at right) to pull a rectangle around the signature. The size of the rectangle is not important, since it will be automatically scaled by BViewer to fit in the designated space on the report form. This space is about six times as wide as it is high. Ideally your rectangular selection region should have roughly these proportions of width to height.

For later versions of Windows it may suffice to merely press the key after selecting the rectangular region of your image,to perform the cropping.

For older version of Windows, from the "Edit" menu item, select "Cut". This saves the selected rectangular portion of the image. From the "Image" menu item, select "Attributes". Resize the image to 10 pixels wide and 10 pixels high. This collapses most of the screen real estate taken up by the full-page image. From the "Edit" menu item, select "Paste". This will paste your cropped signature image back into the Paint program.

Step 4: Save the signature image as a bitmap file.

Successful completion of Step 3 should leave you with an image in the Paint program that looks something like the one to the right.

Next, click to drop down the File menu, as shown. Then click on the Properties item: (shown highlighted).

This causes the Properties window to appear. Make sure it is set to Color and not to Black and White. (24-bit colored bitmap files are the most commonly occurring, and are therefore the least likely to have format-related problems in Windows.)

From the "File" menu item, select "Save As" and select "BMP Picture" as the type of file to save.

The signature file needs to be saved in the "Signatures" folder of the BViewer installation. The illustration below shows where this folder is likely to reside on your computer hard drive.

You must be careful to name the file correctly. Otherwise, BViewer will not be able to find it. If, for example, in the Set Up BViewer tab you entered "Smith" for the Reader Identification, "Last Name", then the name you would choose for your signature file would be SmithSignature. The file type you must choose for saving is ".bmp". Thus, the full file specification for your saved signature file would be SmithSignature.bmp.

If your Reader Identification Last Name were Anderson, then your signature file specification would be AndersonSignature.bmp.
