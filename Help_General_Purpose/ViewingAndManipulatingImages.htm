Viewing and Manipulating Images

Selecting a Standard Image for Viewing

You activate a standard image by clicking on its button on the Standard Image Selector Panel. This causes the selected standard image to be displayed in the Standard Image Display Window.

Standard reference images can be magnified and panned, but no windowing adjustments are supported, since this would compromise their display as a reference standard.

Selecting a Subject Study Image for Viewing

To select the study image you wish to view, you must first activate the study selection list by clicking on the "Select Subject Study" tab near the top of the control panel. With the study selection list visible in the control panel window, left-click your mouse on the desired item of the study selection list. When you click on an item, the associated image is displayed in the Subject Study Display Window, and BViewer automatically advances to the "Enter Interpretation" screen.. (This automatic advance feature can be disabled if needed: Ask your NIOSH facilitator if you prefer to disable auto advance..)

Composing A Study

It may sometimes happen that you are presented with an undigitized film for interpretation or else you have a digital image that BViewer us unable to display, but which can be displayed using another program. In these circumstances where you are viewing the image outside BViewer, you can still perform an interpretation using BViewer.

To do this, click on the "Enter Data to Compose a Study" button. This will result in a popup window where you can enter data for the study. When you exit this window a new study will appear as if it had been imported. Click on the bright green "Add Newly Arrived Images" button when it appears, and the new study will be added to the study selection list. If you click on it, you will be advanced to the interpretation screen where you can proceed with your reading as if a BViewer image were available.

Adjusting the Image Display

You can adjust the image display by using the mouse and by using the control buttons
at the top of the image window:

How to Change the Image Magnification

Position the mouse pointer over the image. Push the mouse wheel away from you to increase the magnification. Pull it toward you to decrease the magnification.

The Invert Colors Button

Clicking the mouse on the Invert Colors button causes black to become white and white to become black, etc. This essentially creates a photographic "negative" of the image. Clicking the button again returns the image to its original coloration.

The Rotate Image Button

Clicking the mouse on the Rotate Image button causes the image to rotate 90 degrees counterclockwise each time the button is clicked.

The Flip Vertically Button

Clicking the mouse on the Flip Vertically button causes the image to be vertically inverted, so that the top becomes the bottom and the bottom becomes the top.

The Flip Horizontally Button

Clicking the mouse on the Flip Horizontally button causes the image to be horizontally inverted, so that the left edge becomes the right edge and the right edge becomes the left edge.

The Adjust To Full Size Button

Clicking the mouse on the Adjust To Full Size button causes the image magnification to be set to match the actual size of the corresponding film image. Clicking the button again, returns the image to the "best fit" size.

NOTE: In order for this resizing to be accurate, the display monitor screen dimensions must be correctly entered on the Set Up BViewer screen. You must measure the width and height in millimeters of your screen display area and enter the values in the appropriate places on the Set Up BViewer screen.

How to Adjust Window Center / Window Width

The apparent brightness and contrast of the subject study image can be adjusted by changing the settings of the window width and window center parameters in the conventional way.

Position the mouse pointer over the central part of the image. Hold down the right mouse button to engage the window center/window width adjustment. Moving the mouse upward increases the window center setting. Moving the mouse to the right increases the window width setting. You can drag the mouse around in a circular motion with the right mouse button depressed in order to "home in" on an optimum setting. The window center and window width setting will be updated in the image window header above the image.

If it is more convenient than moving the mouse around with the right button depressed, you also have the option of entering the desired window width and window center values in the edit boxes.

You can also use the Save Window Settings button to save a particular windowing specification for later use. Later you can click the Apply Saved Window Preset button to restore the specified windowing specifications. This can be especially useful when you have multiple studies being received from the same institution with the same imaging characteristics.

NOTE: If the linear measurement tool is active, then the right mouse button is used for that and is not functional for brightness and contrast changes. You can restore the right mouse button for controlling the brightness and contrast by clicking the Stop Measuring button.

How to Adjust Gamma

The gamma correction for displaying a particular image on a particular display monitor can be set by entering a gamma value in the small edit box.

For a properly calibrated image displayed on a calibrated monitor, the value of gamma should be 1.0.

For an image displayed on an uncalibrated monitor, a value of gamma in the neighborhood of 2.2 may be an appropriate adjustment for the display monitor's luminosity behavior.

(Technical note: The gamma adjustment is a way of compensating for the nonlinear luminosity behavior of display hardware. The gamma setting interacts closely with windowing brightness and contrast settings. A calibrated display device is already gamma-adjusted and does not need any gamma adjustment.

In some circumstances a gamma setting such as 2.2 (common for ordinary color display monitors) may have been applied to an image at its point of origin before it was stored and transmitted. Gamma corrections should never be applied to a stored image, since this correction applies to the specific display device where the image is being viewed. The intent would have been to preset the image to a typical gamma correction for uncalibrated viewing. When viewing such an image on a calibrated monitor, you can undo this pre-stored adjustment by selecting a fractional gamma value less than 1.0. In a well-controlled and well-calibrated image generation and viewing environment, you should never have to make gamma adjustments. A value of 1.0 conforms to properly calibrated hardware.)

The Hide Study Info Button

Left clicking the mouse on the Hide Study Info button removes the text overlay displayed in the upper left corner of the image. This information is extracted from the Dicom header of the image file and provides details such as the subject name, ID, date of birth, etc. Clicking this button a second time will toggle the display of this information back on.

The Reset Image Button

Left clicking the mouse on the Reset Image button restores all the image settings that you may have changed back to their original values. The image is redisplayed as it was when you originally selected it.

The Save Window Settings Button

Left clicking the mouse on the Save Window Settings button causes your current display windowing settings for this image to be saved as a named preset. Saved windowing presets can be reapplied to the current image or applied to any images you view in the future. Clicking this button results in the appearance of the popup window shown below:

Apply Saved Window Preset Button

Left clicking the mouse on the Apply Saved Window Preset Button pops up a window asking you to select from among your saved windowing presets the one you wish to apply to the current image. This window also supports deleting saved presets you no longer need to retain.

Using the Measurement Tool

Left clicking the mouse on the Measure Distance button activates the measurement tool and causes the three measurement buttons to light up, indicating that the measurement process is active. Also, the label on this button is changed to "Stop Measuring". Clicking on this button again toggles the measurement tool off.
Whenever the measurement tool is activated, it takes control of the right mouse button over the image. You can use the right mouse button to measure the distance between two points as follows:
  • Move the mouse to the point on the image where you want to start the measurement.
  • Press the right mouse button and hold it down while dragging the mouse to the point where you want the measurement to end.
  • Release the right mouse button.

Releasing the right mouse button at the end of the distance measurement interval concludes the measurement. The distance in millimeters between the starting and ending points is displayed alongside the measurement.

NOTE: If the linear measurement tool is active, then the right mouse button is used for that and is not functional for window center/window width changes. You can restore the right mouse button for controlling the windowing by clicking the Stop Measuring button.

Left clicking the mouse on the Erase Measurements button will cause all the measurements currently displayed to be erased.

Measurement values are automatically compensated for image magnification changes. By default the distance measurement is calibrated to the assumed film size and the size of the monitor display screen entered on the Set Up BViewer screen. The measurements represent distances on the "film" and not necessarily distances on the subject of the exposure.

If you have some way of providing a better estimate of the actual distances, you can click on the Calibrate Last Measurement button after having specified a measurement of a known distance on the image.
You will be prompted to enter the correct distance in millimeters.
All measurements made previously or subsequently on the current image will be calibrated to this distance.
