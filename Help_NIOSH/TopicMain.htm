What Is BViewer?

BViewer is a computer program used by B Readers to interpret digitized radiographic images according to the ILO guidelines. BViewer can be set up to operate in any one of three supported interpretation environments:

  • General Purpose. This is the default mode of operation when you install BViewer.
  • NIOSH: The mode is used by NIOSH and others specifically for their occupational lung disease monitoring program.
  • Test: This mode is used for administering B Reader certification and recertification examinations.
Your NIOSH facilitator can configure the BViewer interpretation mode that is appropriate for your installation.

During your interpretation, you may frequently need to move your mouse from one display window to another. A quick way to do this is to:

  • Press and release the (escape) key (usually at the upper left of the keyboard). This arms the escape to another screen.
Then press one of the following keys to specify to which screen you wish to move the mouse:
  • r: Move the mouse to the reference standard image display.
  • s: Move the mouse to the subject study image display.
  • c: Move the mouse to the control panel.
  • i: Move the mouse to the interpretation report window.
Pressing any other key will cancel the escape request. This escape mechanism can also be used as a handy way to toggle between the current reference image and the subject study image in the unusual situation that there is only one high-resolution image display device available.

Computer Hardware:

The BViewer software is designed to run on a personal computer running Windows version XP or later. To achieve good performance, the computer should have a reasonably fast processor and plenty of memory. It will need a hard drive with enough storage capacity to retain a minumum of several days worth of subject study images.

The BViewer program should be run on a computer system with three display monitors:

  • Primary Monitor: The Windows desktop monitor for displaying the Control Panel, the Standard Image Selector Panel and the Report Display Window
  • Image Display Monitor: A calibrated high-resolution grayscale monitor for displaying the Standard Image Display Window
  • Image Display Monitor: A calibrated high-resolution grayscale monitor for displaying the Subject Study Display Window
The two image display monitors should be arranged side-by-side to facilitate comparing the subject study image with the various standard images.

BViewer Software:

The BViewer software consists of two separate components, as shown:
The BRetriever component of the BViewer software runs in the background as a Windows service. It runs whenever your computer is turned on, so it can receive image files even when you are not running BViewer. BRetriever receives Dicom image files and processes them to extract the image and to extract the associated information describing the subject, the study and the image. BRetriever repackages this information and presents it to BViewer. Each newly received study is then represented by an entry inserted in the BViewer study selection list, from which it can be selected for display and interpretation.
